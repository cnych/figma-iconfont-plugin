GitHub About

Iconfont is a vector icon management and communication platform created by Ali MUX. The designer uploads the icon to the Iconfont platform, and the user can customize to download the icon in multiple formats. The platform can also convert the icon into a font, which is convenient for front-end engineers to adjust and call.

ICONFONT FIGMA PLUGIN

This figma plugin allows you to directly search for icons in iconfont and insert the icon in svg format directly into your design. It is very simple and convenient.
